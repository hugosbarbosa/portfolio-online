Sobre Mim

Olá! Meu nome é Hugo, sou Desenvolvedor Front-end. Minha paixão é codificar interfaces web modernas e responsivas, e meu stack principal é composto por HTML, CSS, Sass, Bootstrap, JavaScript e React. Possuo um conhecimento complementar valioso em C#, AWS e MySQL, que me permite uma visão mais completa do desenvolvimento. Trabalho de forma organizada, utilizando Git/GitHub para versionamento e adotando a metodologia ágil Scrum e boas práticas de código limpo. Sou um profissional em transição de carreira, trazendo o background de Analista de TI para uma abordagem eficiente no desenvolvimento de software.
